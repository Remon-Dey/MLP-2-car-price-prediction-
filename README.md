# Ford Car Price Prediction using Linear Regression

This project focuses on predicting the selling price of Ford vehicles using Machine Learning. The workflow includes comprehensive data preprocessing, exploratory data analysis, feature engineering, and model training to build an accurate car price prediction system.

## Project Workflow

* Exploratory Data Analysis (EDA)
* Data Cleaning and Preprocessing
* Missing Value Handling
* Outlier Detection and Treatment
* Label Encoding
* One-Hot Encoding
* Feature Engineering and Extraction
* Data Transformation and Scaling
* Linear Regression Model Training
* Model Evaluation and Performance Analysis

## Dataset Features

The dataset contains various Ford car attributes such as:

* Model
* Year
* Transmission Type
* Mileage
* Fuel Type
* Tax
* MPG (Miles Per Gallon)
* Engine Size
* Price (Target Variable)

## Objective

The primary objective of this project is to predict Ford car prices based on vehicle characteristics. By applying proper data preprocessing techniques and training a Linear Regression model, the project demonstrates an end-to-end Machine Learning workflow, from raw data preparation to price prediction and model evaluation.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Results

The Linear Regression model was trained on the processed dataset and evaluated using standard regression metrics to measure prediction performance and accuracy.
